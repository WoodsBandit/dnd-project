## Getting Started

1. **New Campaign**: Create folder in `campaigns/[campaign-name]/`
2. **New Print**: Add STL to appropriate `3d-prints/` subfolder
3. **New Tool**: Build in `tools/` with documentation
4. **New Agent**: Configure in `agents/` with prompt file
