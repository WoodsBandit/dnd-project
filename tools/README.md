# Campaign Tools

## Subfolders

- **generators/** - Random content generators (NPCs, loot, encounters)
- **trackers/** - Game state trackers (initiative, HP, inventory)
- **calculators/** - Math helpers (CR, XP, encounter balance)

## Tool Development Guidelines

- Python preferred for scripts
- Include CLI interface for quick use
- Document inputs/outputs
- Make campaign-agnostic where possible
- Use JSON for data files
