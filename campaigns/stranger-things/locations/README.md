# Locations - Hunt for the Thessalhydra

This folder contains location files for the Stranger Things D&D Starter Set adventure.

## Major Locations

### Starting Area
- `sir-tristans-throne-room.md` - Where the adventure begins

### Journey Locations
- `the-road.md` - Travel section with NPC encounters

### Dungeons
- `troglodyte-caverns.md` - First major dungeon (11 areas)
- `cursed-labyrinth.md` - Random path maze with the Lost Knight

### Otherworldly
- `the-upside-down.md` - The alternate dimension (Demogorgon encounter)
- `thessalhydra-lair.md` - Final boss location

## Organization

Locations are ordered roughly by when they appear in the adventure:
1. Throne Room (Quest start)
2. The Road (Travel/roleplay)
3. Troglodyte Caverns (Dungeon 1)
4. Cursed Labyrinth (Dungeon 2)
5. The Upside Down (Transition zone)
6. Thessalhydra's Lair (Final dungeon)

## Meta Location
- `hawkins-indiana.md` - Background on the 1980s setting inspiration from the show
