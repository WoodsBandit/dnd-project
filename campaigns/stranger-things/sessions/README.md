# Sessions - Hunt for the Thessalhydra

This folder contains session notes and chapter breakdowns for the Stranger Things D&D Starter Set adventure.

## Adventure Structure

The Hunt for the Thessalhydra is designed for 2-4 sessions of play. The adventure breaks down into the following chapters:

### Chapter Files
- `chapter-01-the-quest-begins.md` - Throne room, quest assignment
- `chapter-02-rumors-on-the-road.md` - Travel section, NPC encounters
- `chapter-03-troglodyte-caverns.md` - First dungeon (11 areas)
- `chapter-04-cursed-labyrinth.md` - Maze section, Lost Knight riddles
- `chapter-05-the-upside-down.md` - Demogorgon encounter
- `chapter-06-thessalhydra-lair.md` - Final boss battle

### Session Planning
- `session-00-setup.md` - Pre-game preparation
- `session-log-template.md` - Template for logging actual sessions

## Estimated Pacing

| Chapter | Estimated Time |
|---------|----------------|
| Ch 1: Quest Begins | 15-30 minutes |
| Ch 2: Rumors on the Road | 30-45 minutes |
| Ch 3: Troglodyte Caverns | 1-2 hours |
| Ch 4: Cursed Labyrinth | 45 minutes - 1 hour |
| Ch 5: The Upside Down | 30-45 minutes |
| Ch 6: Thessalhydra's Lair | 45 minutes - 1 hour |

**Total Estimated Time:** 4-6 hours

## Suggested Session Breaks

### 2-Session Format
- **Session 1:** Chapters 1-3 (Quest through Caverns)
- **Session 2:** Chapters 4-6 (Labyrinth through Final Boss)

### 3-Session Format
- **Session 1:** Chapters 1-2 (Quest and Travel)
- **Session 2:** Chapter 3 (Troglodyte Caverns)
- **Session 3:** Chapters 4-6 (Labyrinth to Finale)

### 4-Session Format
- **Session 1:** Chapters 1-2
- **Session 2:** Chapter 3
- **Session 3:** Chapters 4-5
- **Session 4:** Chapter 6 (Extended finale)
