## Print Recommendations by Size

### Small Creatures (Goblins, Kobolds)
```
Scale:           28mm
Layer Height:    0.08mm (resin) / 0.12mm (FDM)
Infill:          0% (hollow) or 10%
Supports:        Light, auto-generated
Orientation:     Upright when possible
Print Time:      15-30 min each (resin)
Batch:           Fill plate, print overnight
```

### Medium Creatures (Orcs, Skeletons)
```
Scale:           28-32mm
Layer Height:    0.05-0.08mm (resin) / 0.12mm (FDM)
Infill:          10-15%
Supports:        Medium density
Orientation:     Slight tilt for weapons
Print Time:      30-60 min each (resin)
```

### Large Creatures (Ogres, Trolls, Young Dragons)
```
Scale:           ~50mm tall
Layer Height:    0.05mm (resin) / 0.12-0.16mm (FDM)
Infill:          10-15%
Supports:        Consider splitting model
Orientation:     Wings/arms may need separate print
Print Time:      2-4 hours (resin)
```

### Huge Creatures (Giants, Adult Dragons)
```
Scale:           ~75-100mm tall
Layer Height:    0.05-0.08mm (resin) / 0.16mm (FDM)
Infill:          5-10%
Supports:        Heavy, tree supports for FDM
Orientation:     Multi-part assembly
Print Time:      4-8+ hours
Notes:           Consider resin for detail, FDM for speed
```

### Gargantuan Creatures (Ancient Dragons, Tarrasque)
```
Scale:           100mm+ tall
Layer Height:    0.08mm (resin) / 0.20mm (FDM)
Infill:          5%
Supports:        Minimal internal, strategic external
Orientation:     Multi-part (body, wings, tail, head)
Print Time:      12-24+ hours total
Notes:           Display pieces, may exceed build plate
```

## Starter Collection (Budget Priority)

If building from scratch, print in this order:

### Wave 1: Immediate Essentials
1. **6-8 Goblins** - Most common low-level enemy
2. **6 Skeletons** - Universal undead
3. **4 Orcs** - Versatile humanoids
4. **4 Wolves** - Common beast encounters
5. **1 Ogre** - First "big" enemy

## Painting Priority

### Quick Tabletop Ready (Speedpaint method)
1. Prime black or grey
2. Zenithal highlight (white from above)
3. Single coat of Speedpaint/Contrast paint
4. Quick drybrush highlights
5. Basing
6. Matte varnish

**Time per mini:** 10-20 minutes

### Display Quality
1. Prime, zenithal
2. Base coat all colors
3. Shade/wash recesses
4. Layer highlights
5. Edge highlighting
6. Details (eyes, gems, etc.)
7. Basing with materials
8. Varnish (gloss for eyes/slime, matte for rest)

**Time per mini:** 2-4+ hours
