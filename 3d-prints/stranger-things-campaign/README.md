## Print Recommendations

### For Miniatures (Phases 1-4)

**Resin (Recommended):**
- Layer height: 0.03-0.05mm
- Hollow with 1.5mm walls
- Add 2.5mm drain holes
- Orient at 30-45 degrees
- Light supports on faces

**FDM (Budget Option):**
- Layer height: 0.08-0.12mm
- 0.2mm nozzle recommended
- Print weapons separately
- Use tree supports

### For Terrain (Phases 5-8)

**FDM (Recommended):**
- Layer height: 0.16-0.2mm
- 10-15% infill
- No supports for most pieces
- PLA or PLA+ material

### For Props (Phase 9)

**Either Print Method:**
- Small items: Resin for detail
- Display items: FDM for size
