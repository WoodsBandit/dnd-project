# 3D Prints

## Subfolders

- **miniatures/** - Character and monster minis (28mm standard)
- **terrain/** - Dungeon tiles, buildings, landscapes
- **props/** - Items, weapons, accessories

## File Naming Convention

```
[type]-[name]-[scale].stl
[type]-[name]-[scale].gcode
[type]-[name]-[scale].md  (print settings)
```

Examples:
- `goblin-archer-28mm.stl`
- `dungeon-tile-2x2.stl`
- `treasure-chest-28mm.stl`

## Print Settings Template

Create a `.md` file alongside each STL with:
- Layer height
- Infill percentage
- Support requirements
- Estimated print time
- Material recommendations
